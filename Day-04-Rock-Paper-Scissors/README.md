# Day 4 - Rock Paper Scissors ✊✋✌️

Classic Rock Paper Scissors game against the computer.

## What I Learned
- Lists and indexing
- Random module (`random.choice`)
- Nested if statements
- ASCII art for game visuals
- Basic game loop logic

## How to Play
Run `main.py` → Choose rock (0), paper (1), or scissors (2) → See if you beat the computer!
